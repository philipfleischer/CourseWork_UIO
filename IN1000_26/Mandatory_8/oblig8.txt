Oppgave 2A: Prosedyre
Skriv en prosedyre 'godkjent' som sjekker om en student kan fremstille seg til eksamen i IN1000. Maks. poeng for obligene i IN1000 representerer vi i denne lista: [3,5,5,5,5,6,1,1]. Maks poeng er 3 for oblig 1, 5 for oblig2, osv. På de første 6 obligene kan poengene variere fra 0 til maks, men for å få godkjent de første seks må summen være minst 19 (maks sum er 29). På de to siste gis 1 for godkjent og 0 for ikke godkjent. For at det obligatoriske arbeidet skal være godkjent må summen av poengene på oblig 1-6 være minst 19, og oblig 7 og 8 må begge ha 1 poeng. 'godkjent' tar poenglista som eneste parameter og skal skrive ut "Obligkravet er oppfylt" om obligkravet er godkjent. Hvis det ikke er godkjent, skal prosedyren skrive ut alle kravene som ikke er ok. De tre kravene er altså:
- minst 19 poeng på oblig 1-6
- mer enn 0 poeng på oblig 7
- mer enn 0 poeng på oblig 8
Skriv proseddyren godkjent her:

def godkjent(poengliste):
    if len(poengliste) != 8:
        print("Ugyldig antall poeng for innleveringer!")
        return

    godkjent_grense = 21
    total_poeng = 0
    oblig7 = False
    oblig8 = False

    for i in range(poengliste):
        if i == 0:
            if poengliste[i] < 0 or poengliste[i] > 3:
                print(f"Ugyldig poengverdi for oblig {i+1} med poeng: {poengliste[i]}!")
                return
            else:
                total_poeng += poengliste[i]
        if i > 0 and i < 4:
            if poengliste[i] < 0 or poengliste[i] > 5:
                print(f"Ugyldig poengverdi for oblig {i+1} med poeng: {poengliste[i]}!")
                return
            else:
                total_poeng += poengliste[i]
        if i == 5:
            if poengliste[i] < 0 or poengliste[i] > 6:
                print(f"Ugyldig poengverdi for oblig {i+1} med poeng: {poengliste[i]}!")
                return
            else:
                total_poeng += poengliste[i]
        if i == 6:
            if poengliste[i] == 1:
                total_poeng += 1
                oblig7 = True
            else:
                print(f"Ikke godkjent Obligatorisk oblig {i+1}"!)
                return
        if i == 7:
            if poengliste[i] == 1:
                total_poeng += 1
                oblig8 = True
            else:
                print(f"Ikke godkjent Obligatorisk oblig {i+1}"!)
                return

    if total_poeng < 21 or total_poeng > 31:
        print(f"Total poengsum {total_poeng} overskrider maks total poengsum på 31")
    elif not oblig7 or not oblig8:
        print("Alle obligatoriske oppgaver er ikke bestått")
    else:
        print("Obligkravet er oppfylt")

def main():
    poengliste: [3,5,5,5,5,6,1,1]
    godkjent(poengliste)

if __name__ == "__main__":
    main()

class Dato:
    def __init__(self, dd, mm, aaaa):
        self._dagnr = dd
        self._mnd = mm
        self._år = aaaa

    def dagnr(self):
        return self._dagnr
    def mnd(self):
        return self._mnd
    def år(self):
        return self._år

I tillegg har vennen din skrevet denne funksjonen som lager en kalender for året 2023 som ei liste av Dato-objekter:

def lagKalender2023():
    kalender = []
    for dag in range(32):
        for mnd in range(13):
            kalender.append(Dato(dag,mnd,2023))
    return kalender

Nå ber vennen din deg om å hjelpe til med å skrive en funksjon som fjerner dato-objekter som ikke finnes. Funksjonen tar inn ei liste med datoobjekter (som er det lagKalender2023 returnerer) og returnerer lista der de ugyldige datoene er fjernet. For å gjøre dette kan du legge til grunn at denne ordboken er tigjengelig i det globale skopet:

dageriMnd = { 1:31, 2:28, 4:30, 5:31, 6:30, 7:31, 8:31, 9:30, 10:31, 11:30, 12,31 }

Skriv funksjonen fjernUgyldigeDatoer her:

def fjernUgyldigeDatoer(kalender):
    maks_dag_verdi = 31
    for key, value in dageriMnd.items():
        if value == 30:
            kalender.remove(Dato(maks_dag_verdi, key, 2023))
        if value == 28:
            for i in range(3):
                kalender.remove(Dato(maks_dag_verdi - i, key, 2023))
    return kalender

I tillegg til ordboken dageriMnd har du en global liste over skuddår i perioden:

dageriMnd = { 1:31, 2:28, 3:31, 4:30, 5:31, 6:30, 7:31, 8:31, 9:30, 10:31, 11:30, 12:31 }

Ordaboken som returneres skal ha årstallet som nøkkel, og innholdsverdien skal være kalenderen (liste av datoobjekter) for det året.

Skriv funksjonen lagKalender2023_2040 her:

Svar ->

def lagKalender2023_2040():
    kalender = []
    for key, value in dageriMnd.items():
        kalender.extend(kalenderHjelper(key, value))

    return kalender

def kalenderHjelper(mnd, dager):
    liste = []
    current_aar = 2023
    for dag in range(dager):
        liste.append(Dato(dag,mnd,current_aar))

    if mnd == 2 and current_aar in skuddår:
        liste.append(Dato(29,mnd,current_aar))

    current_aar += 1

    return liste

def main():
    antall_aar = 2040-2023
    kalender = []
    for i in range(antall_aar+1):
        kalender = lagKalender2023_2040()

    print(kalender)
