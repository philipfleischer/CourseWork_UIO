Oppgave 3a)
Skriv funksjonen 'jages' som har en parameter 'dyreliste' som er en liste med dyr, hvor hver dyr er en av strengene 'mus', 'katt' eller 'hund'. Hvert dyr kan være flere ganger i listen.

Hvis noen av dyrene i listen vil jage noen av de andre dyrene skal funksjonen returnere True, ellers False. Vi antar at en hund alltid vil jage en katt, og at en katt alltid vil jage en mus, men at dyrene ellers ignorerer hverandre.

Kallet jages(['mus', 'katt']) skal altså returnere True fordi katten vil jage musen, mens kallet jages(['mus', 'mus', 'hund']) skal returnere False.

Skriv ditt svar her:

def jages(dyreliste):
    if "mus" and "katt" in dyreliste:
        return True
    elif "katt" og "hund" in dyreliste:
        return True
    return False

def main():
    dyr1 = ['mus', 'katt']
    dyr2 = ['katt', 'mus']
    print(jages(dyr1))
    print(jages(dyr2))

if __name__ == "__main__":
    main()

Funksjonen utvidet_jages har to parametere: en parameter dyreliste som er en liste av strenger, og en parameter jaging som er en ordbok hvor hver nøkkel er et dyr (en streng) som jager dyret (streng) i den tilhørende innholdsverdien. Denne funksjonen kan håndtere alle slags dyr.

Kallet utvidet_jages(['ulv', 'mus', 'sau'], {'ulv':'sau'}) skal altså returnere True fordi vi har definert at ulv jager sau og begge disse dyrene finnes i dyrelisten.

Skriv ditt svar her:

def utvidet_jages(dyreliste, jaget):
    for jager, jages in jaget.items():
        if jager in dyreliste and jages in dyreliste:
            return True
    return False

def main():
    print(utvidet_jages(['ulv', 'mus', 'sau'], {'ulv':'sau'}))
    print(utvidet_jages(['ulv', 'mus', 'hund'], {'ulv':'sau', 'katt':'mus'}))

if __name__ == "__main__":
    main()


Oppgave 3c)
Skriv en funksjon 'flertall' som tar inn en liste dyreliste med navn på ulike dyr (en liste av strenger), og som returerer navnet på dyret det finnes flest av i listen (hvilken streng som finnes flest ganger i listen). Dersom det er likt mellom to - altså at de to dyrene det finnes flet av er til stede like mange ganger i listen - så skal funksjonen returnere "uavgjort".

Kallet flertall(['ulv', 'sau', 'ulv']) skal returnere 'ulv', mens kallet flertall(['ulv', 'sau', 'sau', 'ulv']) skal returnere 'uavgjort'.

Skriv ditt svar her:

def flertall(dyreliste):
    dyr_ordbok = {}

    for dyr in dyreliste:
        if dyr not in dyr_ordbok:
            dyr_ordbok[dyr] = 1
        dyr_ordbok[dyr] += 1

    dyr_oftest = None
    dyr_flest = 0
    uavgjort = False

    for dyr in dyr_ordbok:
        current_antall = dyr_ordbok[dyr]

        if current_antall > dyr_flest:
            dyr_oftest = dyr
            dyr_flest = current_antall
            uavgjort = False
        elif current_antall == dyr_flest:
            uavgjort = True

    if uavgjort:
        return "Uavgjort"
    else:
        return dyr_oftest

def main():
    print(flertall(['ulv', 'sau', 'ulv']))      # 'ulv'
    print(flertall(['ulv', 'sau', 'sau', 'ulv']))     # 'Uavgjort'

if __name__ == "__main__":
    main()

Skriv en funksjon felles som har en parameter 'tall_lister' som tar inn en nøstet liste - en liste av lister av heltall. Funksjonen skal returnere en liste med alle heltall som finnes to eller flere ganger i den nøstede listen. Det er uten betydning om et slikt heltall finnes i flere ulike lister i den nøstede listen eller om der finnes flere ganger i den samme lsiten. Det er også uten betydning hvilken rekkefølge de felles tallene returneres i.

Kallet felles([ [1,5,1], [3,4], [2,3] ]) skal altså returnere enten [1,3] eller [3,1] siden tallene 1 og 3 finnes to steder i den nøstede listen (og siden rekkefølgen på tallene som returneres ikke er av betydning).

Skriv ditt svar her:

def fells(tall_lister):
    for_ordbok = {}
    for liste in tall_lister:
        for tall in liste:
            if tall in for_ordbok:
                for_ordbok[tall] += 1
            else:
                for_ordbok[tall] = 1
    ny_liste = []
    for tall, antall in for_ordbok.items():
        if antall > 1:
            ny_liste.append(tall)

    return ny_liste

def main():
    print(felles([ [1,5,1], [3,4], [2,3] ])) # [1,3] eller [3,1]

if __name__ == "__main__":
    main()

Skriv en funksjon 'adskilt' som har en parameter 'tall_lister' som tar inn en nøstet liste - en (ikke-tom) liste av (ikke-tomme) lister av tall. Funksjonen skal returnere True dersom det i den nøstede listen finnes to lister hvor det er slik at hvert av tallene i den ene listen er ekte større enn alle tallene i den andre listen.

Kallet adskilt([ [1,10], [6,8], [2,3,7] ]) skal returnere True fordi hvert tal i den andre listen er større enn alle tall i den tredje.
Kallet adskilt([ [1,100], [6,8], [2,3,7] ]) skal returnere False.

Skriv ditt svar her:

def adskilt(tall_lister):
    for liste1 in tall_lister:
        for liste2 in tall_lister:
            if min(liste1) > max(liste2):
                return True
    return False

def main():
    print(adskilt([ [1,10], [6,8], [2,3,7] ]))      #True
    print(adskilt([ [1,100], [6,8], [2,3,7] ]))     #False
